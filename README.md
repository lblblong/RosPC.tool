# Ros 的PC端工具
与 RosApi、turtlebot 共同使用。
目前已具备：
- 移动
- 设置锚点
- 扫图
- 编辑地图

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
